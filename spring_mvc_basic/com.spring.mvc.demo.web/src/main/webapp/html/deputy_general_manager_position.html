销售副总
外部联系：客户、广告商等
直属下级：市场部经理、营销部经理等
  • 岗位职责简述：
  • 1、 销售目标达成；
  • 2、 销售费用控制；
  • 3、 应收款控制；
  • 4、 品牌建设。
  • 5、 销售系统建设。
工作项目 工作内容 完成标准 检查人 考核或赏罚
年度销售目标直接责任 根据公司年度发展计划进行分解 由董事会（总经理）确定年度销售额目标 总经理/财务部 纳入绩效考核
销售费用控制直接责任 根据年度预算，合理分配及严格控制销售费用 由董事会（总经理）确定年度销售费用预算 总经理/财务部 纳入绩效考核
应收款控制领导责任 应收款控制在公司目标范围内
  • 1、按公司要求的应收款控制标准（万元/年）逐月检查控制
  • 2、形成月度分析报告，每月3日前上报
 总经理/财务部 纳入绩效考核
年度营销规划直接责任 依据公司战略及营销策略，负责制定年度营销目标、计划及实施方案 12月31日前按要求提交下年度营销规划详细方案，供年度会议讨论 总经理 未完成乐捐元
制度建设领导责任 建立、健全各项规章制度，推动销售系统管理规范化、科学化
  • 1、1月20日前所有规章制度进行检查，并列明须修订、增加的制度，上报总经理
  • 2、制定修订计划，按计划每月执行
 总经理 未完成乐捐元
销售任务制定直接责任
  • 1、依据整体销售目标，提交销售计划方案并分解下达
  • 2、监督实施销售过程，控制并完成销售任务
  • 1、12月31日前制定下年度销售目标任务计划
  • 2、分解计划到每月，制定跟踪落实计划，月度计划每月3日前上报
 总经理 未完成乐捐元
品牌维护领导责任 制定品牌管理策略，维护公司品牌，塑造企业形象
  • 1、6月30日前完成一次品牌调研，提交分析报告
  • 2、制定改善措施报告，7月3日前上报实施并跟踪
 总经理 未完成乐捐元
新网点拓展领导责任
  • 1、依据销售计划制定网点开发策略，并分解下达
  • 2、监督实施新网点拓展任务，控制并完成网点拓展
  • 1、依据销售计划12月31日前制定年度网点拓展计划
  • 2、分解到每月，制定跟踪落实计划，每月3日前上报
 总经理 未完成乐捐元
临时事务直接责任 完成上级交办的临时任务
  • 1、领导安排事情的及时处理
  • 2、领导交办的临时事务及时跟踪、回复
 总经理 按《执行令》奖罚
