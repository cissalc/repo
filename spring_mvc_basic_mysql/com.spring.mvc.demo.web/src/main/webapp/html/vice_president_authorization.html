字【2013】第1号

副总经理授权书

授权方：
XXXX洁具有限公司总经理 XXX
行权方：
XXXX洁具有限公司副总经理 XXX
授权说明：
公司总经理为支持生产经理工作开展，经公司管理层研究决定自2013年10月份25日起，特授予生产经理下列权力：
授权权力：
授权项目 授权内容 红线规定
人事权
  • 1、 拥有行政部、采购部、生产部、品管部、工艺技术部、开发部、PMC部、工程部、业务部部门经理人事建议权；
  • 2、 拥有以上部门经理以下职务任免及解聘权；
  • 3、 以上部门包括经理在内的管理岗位变动设置的建议权。
  • 1、不能利用职权拉帮结派，与公司利益相冲突。
  • 2、不能为袒护个别干部及私利不辨是非引犯众怒。
财务权
  • 1、 预算内的所辖部门薪资、奖金、计件工价的核定权、审批权；
  • 2、 所辖部门员工计划外费用（元以下）的审批权；
  • 3、 公司年底上述各部门干部奖金的拟定和建议权。
  • 1、为一己私利不能做到公平、公正从而引发大多数人不满。
  • 2、薪资、计件工价审核不严谨导致严重不良后果
事务权
  • 1、 所辖部门干部的日常行为管理权、处置权、奖惩建议权（符合公司制度内的）；
  • 2、 所辖部门之间冲突的调剂权和处理权，但同时保留当事人的申述权和总经理最终裁定权；
  • 3、 应急事务和突发事故的监督权和处理权。
  • 4、 对所辖部门相关的责任事故认定及处理执行权
  • 5、 产品开发及工艺改善的部分决定权和处置权
  • 6、 对新产品开发过程中除外观设计外其他材料、结构等技术因素的决定权
  • 7、 对产品生产工艺的变动、改善的决定权
  • 8、 对生产设备采购的建议权
  • 9、 当确认有供方因品质、服务等重要因素不符合我公司利益时，对其有中断合作的决定权。
  • 10、 对新选定供应商有建议权。
  • 11、 当产品在开发，工艺变动，市场要求等因素变动时导致影响品质要求的变动，对此，有决定品质规范及控制范围的权利。
  • 12、 在客户较紧急的情况下，可直接与客户沟通，并在产品排产顺序、产品特殊生产要求等方面有决定权。
  • 13、 对销售系统的产品技术推广、售后难点等方面有建议权。
  • 1、不能明辨是非做到公平、公正从而导致严重错误。
  • 2、由于拉帮结派导致偏袒护短。
  • 2、技术决定权使用不当给公司造成万元以上经济损失
  • 2、设备采购中收受贿赂。
  • 2、对品质要求把握不当造成万元以上经济损失

一 行权期间要求：

1) 每月以面谈或书面形式向总经理汇报一次所辖部门工作进展情况。
2) 各部门工作指标量化明确，每周例会上报。
3) 审核所辖部门各项制度建设及执行情况。
4) 支持和配合执行公司临时规定的相关制度。
5)每月季末日组织召开质量例会会一次。

二 地雷区：

1）受权人嫉贤妒能，利用职务排挤人才，经查实，首次给予警告和罚款元；
2）受权人徇私舞弊，中饱私囊，拿回扣等经查实， 元以内，扣除分红；超出元以上，自动解除合同；情节相当严重者，追究相关法律责任。

三 双方承诺：

1)授权人承诺：破坏授权一次乐捐 元，两次以上重议授权书。
2)行权人承诺：违规行使权力一次乐捐 元，两次以上调离本职岗位。

授权期限：

本授权有效期为此授权书签发之日起至止。
