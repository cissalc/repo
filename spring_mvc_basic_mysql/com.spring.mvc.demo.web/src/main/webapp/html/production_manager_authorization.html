字【2014】第1号

生产经理授权书

授权方：
总经理
行权方：
生产经理
授权说明：
公司总经理为支持生产经理工作开展，经公司管理层研究决定自2014年10月份25日起，特授予生产经理下列权力：
授权权力：
授权项目 授权内容 红线规定
人事权
  • 1、拥有注塑部，品质部及生管部人事建议权；
  • 2、拥有注塑部，品质部及生管部主管任免及解聘权；
  • 1、不能任人唯亲；
  • 2、所有聘任需按公司流程及标准进行考核上岗。违反相关程序任免所辖部门干部。乐捐料（元/次
财务权
  • 1、预算内的所辖部门薪资、奖金建议权；
  • 2、所辖部门辅料（元以下）请购审批权；
  • 3、所辖部门员工差旅费（元以下）的审批权；
  • 4、公司年底上述各部门干部分红的拟定和建议权。
  • 5、所辖部门人员的薪酬标准建议权。
  • 不能越权、越限审批。若有违反乐捐元/次
事务权（工作权）
  • 1、 所辖部门干部的日常行为管理权、处置权、奖惩建议权（符合公司制度内的）；
  • 2、 产品品质最终判定权；
  • 3、 生产材料选择建议权；
  • 4、 出货车辆调度权 ；
  • 5、 生产排程调度权；
  • 6、 设备保养和维修鉴定审批权；
  • 7、 安全生产防范措施决定权和违规操作处罚权；
  • 8、 所辖部门之间冲突的调剂权和处理权；
  • 9、 应急事务和突发事故的监督权和处理权。
  • 10、 对所辖部门相关的责任事故认定及处理执行权
  • 1、流出产品出现品质问题导致公司元以上损失，承担实际损失的20%
  • 2、违规调度车辆，公车私用者乐捐元/次；
  • 3、因违规操作导致安全事故取消当月奖金。

一 行权期间要求：

1) 每周一以例会形式向总经理汇报一次所辖部门工作进展情况。
2) 生产效率、生产达成率、损耗率、生产合格率、成本控制下降率，机器稼动率于书面形式每周一上报一份。
3) 审核所辖部门员工异常出勤情况准时提交给行政部。
4) 支持和配合执行公司临时规定的相关制度。
5) 每天下午17：00召集所辖部门主管人员开展生产协调会。

二 地雷区：

1）行权方嫉贤妒能，利用职务排挤人才，经查实，首次给予警告和罚款元；
2）行权方徇私舞弊，中饱私囊，拿回扣等一经查实，自动解除合同；情节相当严重者，追究相关法律责任。

三 双方承诺：

1)授权人承诺：破坏授权一次乐捐 元，两次以上重议授权书。
2)行权人承诺：违规行使权力一次乐捐 元 或按照红线规定乐捐，两次以上调离本职岗位。
3) 审核所辖部门员工异常出勤情况准时提交给行政部。
4) 支持和配合执行公司临时规定的相关制度。
5)每天下午17：00召集所辖部门主管人员开展生产协调会。

授权期限：

本授权有效期为此授权书签发之日起至止。
