生产安全管理
  • 1）负责一分厂安全生产管理；
  • 2)负责组织生产实施；
  • 3)负责生产成本控制；
  • 4)负责生产过程质量控制。
  • 1)每季度开展一次全员安全生产教育；
  • 2)每月二次安全排查；
  • 3)每天个一次巡视现场操作及设备维护情况；
  • 4)工伤事故元以下由主管、厂长支付。

生产组织
  • 1)根据生产计划，掌控生产进度；
  • 2)搞好各车间的协调，组织分配人力资源，平衡调度、设备、材料等；
  • 1)按照生产计划单的要求完成每日计划
 生产总监 月度检查 月考年发
生产过程质量管理
  • 1)按照质量管理制度要求组织过程质量控制；
  • 2)对质量状况进行总结分析
  • 1)定时（每周一次）召开质量
  • 2)产品合格率在以上。
 质检部与生产总监每周检查 月考年发
生产过程成本管理
  • 1)负责控制生产成本，组织并讨论各原材料利用率与技术更新；
  • 1)严格贯彻成本制度并执行；
  • 2)严格控制并检查各原材料利用；
  • 3)成本控制在标准成本范围之内；
  • 4)每月召开一次成本分析会议。

  • 4)员工月工资编制；
  • 5)完成公司临时下达的任务；
  • 1)产品上线后天内完成计件单件的拟定；
