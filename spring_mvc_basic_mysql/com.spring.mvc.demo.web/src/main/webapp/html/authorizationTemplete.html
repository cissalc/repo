字【2014】第1号

生产厂长管理授权书

授权内容：

经公司管理层研究决定自2014年1月份1日起，任命为一分厂厂长，负责一分厂的生产制造。公司总经理为支持厂长的工作开展，特授予如下行使权力.

授权权力：

1) 生产员工任免权，对所辖车间主管的任免、调动、调薪具有建议权。
2) 生产计件工价制定权和车间内部工价调整权，并将结果按规定上报。
3) 设备维护使用监督权，对员工无故损坏和工作失责处理权。
4) 本车间员工内部冲突和生产过程不良事故处理权，如遇质量争议，同品管部经理协商解决。
5) 需要设计部门支持的调动权。
6) 生产一线员工（含管理人员）出勤监督权和处理权（符合公司制度内的）。
7) 生产过程员工突出贡献奖励批准权（当月次元以内不累计），享有建议权，最终上报总经理或总监审批生效。

行权期间要求：

1) 订单执行进度报表每周以书面形式向生产总监汇报一次。
2) 出货异常情况每周要以书面形式向生产总监和总经理助理汇报各一份。
3) 月度生产计件员工工资分配报表审核确认，并报生产总监审批。
4) 支持和配合执行公司临时规定的相关制度。

授权期限：

本授权有效期为此授权书签发之日起至法人代表书面声明本授权作废为止。

签署：

主题词： 生产厂长 授权书
抄送：销售部、物流部、生产部、技术部、财务部及相关生产车间主任。
