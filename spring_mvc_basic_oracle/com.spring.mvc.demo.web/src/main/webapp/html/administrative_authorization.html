字【2014】第1号

行政人力资源部经理授权书

授权内容：
经公司管委会研究决定自年月日起，任命 为有限公司行政人力资源部经理，负责公司行政人力资源工作。公司总经理为支持该同志工作，特授予如下权力：
授权权力：
授权项目 授权内容 备注
人事权
  • 1、 行政人力资源部下辖人力资源部门，同时下辖运营中心、制造中心、财务部三大部门行政纪律监督工作。
  • 2、 拥有行政人力资源部人事调度权，直接下属的录用及解聘建议权，除直接下属外的录用、解聘权。
  • 3、 四大部门人员的薪酬调整权（在公司核定标准内的）
财务权
  • 1、 预算内的所直辖四部门资金支配和审批权。
  • 2、 预算内部门经费的支配权。
事务权
  • 1、 对四大部门员工的日常行为管理权和处置权（符合公司制度内的）。
  • 2、 部门间冲突的调剂权和处理权。
  • 3 行政人力资源部内协作支持的调动权和其它部门调动的建议权。
  • 4、 应急事务和突发事故的监督权和处理权。
  • 1、不能明辨是非做到公平、公正从而导致严重错误。
  • 2、由于拉帮结派导致偏袒护短。
  • 2、技术决定权使用不当给公司造成万元以上经济损失
  • 2、设备采购中收受贿赂。
  • 2、对品质要求把握不当造成万元以上经济损失

一 行权期间要求：

1) 每周一上午以面谈形式向总经理汇报一次部门工作进展情况。
2) 人员流失、培训、员工工资报表形式向总经理每月上报一份。

二 地雷区：

1）行权方嫉贤妒能，利用职务排挤人才，经查实，首次给予警告和罚款元；
2）行权方徇私舞弊，中饱私囊，拿回扣等一经查实，自动解除合同；情节相当严重者，追究相关法律责任。

三 双方承诺：

1)授权人承诺：破坏授权一次乐捐 元，两次以上重议授权书。
2)行权人承诺：违规行使权力一次乐捐 元 或按照红线规定乐捐，两次以上调离本职岗位。

授权期限：

本授权有效期为此授权书签发之日起至止。

总经理签署：
